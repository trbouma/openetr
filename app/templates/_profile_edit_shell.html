The mnemonic shown in the reveal dialog is not the recommended wallet recovery format for this Taproot flow, because many wallet apps will interpret it as an HD seed and derive a different wallet.
To spend funds, import the revealed taproot_wif into a Bitcoin wallet of your choice that supports Taproot single-address imports. Before sending or spending meaningful funds, confirm that the wallet shows the same p2tr address displayed on this page.
